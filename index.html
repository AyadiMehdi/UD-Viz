UD-Viz is a JavaScript client based on iTowns allowing to visualize, analyse and interact with urban data. It is developed as part of the Urban Data Services and Vizualisation Project.

Examples

  • All Widgets

    This demo implements all widgets and extensions currently available. As of today, this includes :
    - Temporal
    - Help & About
    - Documents
    - Contribute
    - Guided tours
    - Authentication
    - Documents to validate
    - Document comments
    - Geocoding
    - 3DTiles debug
    - Camera positioner
    - Document links

  • Multiple CityGML type visualization on Lyon 1, 2 and 5

    Visualisation of three 3D Tiles Layers representing the buildings, the terrain and the water of the 1st, 2nd and 5th borough of Lyon, extracted from CityGML.

  • WorldMap basemap

    This example use IGN basemap to be able to navigate all over the world

Individual Modules

  • Documents Widget

    In this demo you can consult documents in a document browser and search them using keywords and other fields.

  • Geocoding Widget

    Geocoding allows a user to navigate in the city by searching addresses or place names.

  • Camera Positioner Widget

    The camera positioner is a tool that show the coordinates and the orientation of the camera.

  • 3DTiles Debug Widget

    3DTiles Debug displays a window which shows various information about 3DTiles objects and city objects.

  • City Objects Widget

    This extensions allows the user to explore city objects and color them according to filters.

  • LayerChoice Widget

    This extensions interact with layers

  • Temporal Widget

    This extensions allows the user to explore city objects and color them according to filters.
